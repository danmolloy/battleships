# Battleships

This is an ongoing project of the classic game Battleships, Player vs CPU.

## Stack:
- React
- Redux
- Tailwind
- Jest
- TypeScript
- Mirage.js
- Create React App

## Rules

Ships:
1 x Carrier (5)
1 x Battleship (4)
1 x Destroyer (3)
1 x Submarine (3)
1 x Patrol Boat (2)

Player and CPU take turns attacking.
